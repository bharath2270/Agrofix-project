# AgroFix - Bulk Agri Order Platform

AgroFix is a full-stack web application for bulk ordering of vegetables and fruits. It connects users (buyers) with local produce vendors, enabling smooth order placements, tracking, and admin-level product management.

## Features

###  User Panel
- View available vegetables/fruits
- Place bulk orders
- View order history and status
- Register/Login

###  Admin Panel
- Admin authentication
- Add/Edit/Delete products
- View and manage all orders
- Track user activities

##  Tech Stack

**Frontend**
- React.js (with React Router)
- Bootstrap
- Axios

**Backend**
- Node.js

- Express.js
- MongoDB (via Mongoose)
- JWT for authentication
- Multer (for image uploads)

**Database**
- MongoDB Atlas / Local MongoDB
